# **App Name**: MicroPool

## Core Features:

- Pool Creation: Create new on-chain micro-investment pools with name, description, funding goal, deadline, and purpose.
- Contribution: Contribute ETH or ERC-20 tokens to pools and automatically receive share tokens.
- Proposal Creation: Create proposals to spend funds with title, description, requested amount, and recipient.
- Voting: Vote on proposals using share tokens with token-weighted voting.
- On-Chain Governance: Implement on-chain governance with quorum and majority requirements for proposal execution.
- Pool Listing: Display a list of all available pools on the homepage.
- Smart Contract Deployment Verification Tool: Use generative AI as a tool that allows a developer to verify smart contract deployments and suggest deployment parameters

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to convey trust and security, reflecting the investment nature of the app.
- Background color: Light gray (#F5F7FA), providing a clean and modern backdrop.
- Accent color: Light violet (#B290E3) to highlight important actions and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable user interface.
- Use simple and clean icons to represent different pool categories and actions.
- Design a responsive layout that adapts to different screen sizes, ensuring a consistent user experience across devices.
- Subtle transitions and animations to provide feedback and enhance the user experience.
